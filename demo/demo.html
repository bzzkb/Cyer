Cyerversion1.0.1

Cyer是一个轻量、小巧的js框架，精简易懂的API设计，支持链式调用，有点jQuery的味道。核心部分为选择器(selector)、dom操作、event机制。

  • 简洁易懂的API
  • 兼容主流浏览器Firefox、chrome、IE以及IE6+
  • 支持链式调用
  • 便于处理页面交互效果
  • 暂不支持ajax、animate功能

项目主页: https://github.com/hcy2367/Cyer

Cyer api 简洁易懂，接触过jQuery的同学通过阅读文档5分钟即可上手。

引入文件

在页面 head 引入 Cyer脚本文件:

<script src="source/Cyer.js"></script>

事件机制

  • bind(type, callback)

    概述
    为元素绑定事件(目前只支持单个元素的事件绑定)，参数依次为事件类型、事件处理函数，返回实例对象。
    参数
    String && Function
    示例1：mouseover事件，callback参数是一个匿名函数。

    p#block25

    $("#block25").bind("mouseover", function() {alert("hello world!");});
    示例2：click事件，callback参数是一个函数名(指针)。

    p#block26

     
                             var fun = function() {
                                alert("hello world!");
                            }
                            $("#block26").bind("click", fun);
    示例2：为一个元素同时绑定多个事件，该例子模拟hover事件。由于弹出的弹出框会影响mouseout事件的触发，因此可用consol.log代替alert查看结果。

    p#block27

                                var fun1 = function() {
                                alert("hello world!");
                            };
                            var fun2 = function() {
                                alert("hello world again!");
                            }; 
                            $("#block27").bind("mouseover", fun1).bind("mouseout",fun2);

  • unbind(type, callback)

    概述
    为元素移除事件(目前只支持单个元素的事件移除)，参数依次为事件类型、事件处理函数，返回实例对象。
    参数
    String && Function
    示例：注意callback参数必须是一个函数名(指针)，且bind和unbind的callback参数相同。

    p#block28

                                var func = function() {
                                alert("hello world!");
                            };
                            $("#block28").bind("mouseover", func);
                            $("#block28").unbind("mouseover", func);

  • $.event.add(elem, type, callback)

    概述
    为元素绑定事件(目前只支持单个元素的事件绑定)，参数依次为要绑定的元素、事件类型、事件处理函数，返回实例对象。
    参数
    Object && String && Function
    示例：用法同bind方法一致

    p#block29

    $("#block29").bind("mouseover", function() {alert("hello world!");});

  • $.event.remove(elem, type, callback)

    概述
    为元素移除事件(目前只支持单个元素的事件移除)，参数依次为要处理的元素、事件类型、事件处理函数，返回实例对象。
    参数
    Object && String && Function
    示例：用法同unbind方法一致

    p#block30

                                var func1 = function() {
                                alert("hello world!");
                            };
                            $("#block30").bind("mouseover", func1);
                            $("#block30").unbind("mouseover", func1);

  • $.event.fix(event)

    概述
    获取兼容各浏览器的event事件对象，注意--事件对象event只存在于事件处理函数调用期间。
    参数
    Object
    示例

    p#block31

                                var func2 = function(event) {
                                alert($.event.fix(event) === event); //true
                                alert("事件类型:" + event.type + 
                                "\n被绑定事件的元素:" + event.target);
                            };
                            $("#block31").bind("click", func2);

  • event.preventDefault()

    概述
    取消或阻止浏览器的默认事件
    参数
    Empty
    示例1：没有调用event的preventDefault方法时，超链接将自动跳转。

    百度

    $("#anchor1").bind("click", function() {alert("嘿嘿，我将自动跳转!");});
    示例2：调用event的preventDefault方法时，超链接不会跳转。

    百度

                                $("#anchor2").bind("click", function(event) {
                                event.preventDefault();
                                alert("嘿嘿，我不会跳转!");
                            });

  • event.stopPropagation()

    概述
    取消或阻止事件冒泡
    参数
    Empty
    示例1：没有调用event的stopPropagation方法时，会发生事件冒泡，即事件会传递到body元素并立即触发。

    p#block34

                            var func3 = function() {
                            alert("body is clicked");
                        };
                        //注意：因为其他元素也绑定了click事件，
                        //所以暂时把body的click事件绑定注释了，测试时请把注释去掉。
                        $(document.body || 
                            document.documentElement).bind("click", func3);
                        $("#block34").bind("click", func3);
    示例2：调用event的stopPropagation方法时，不会发生事件冒泡，即事件不会传递到body元素。

    p#block35

                            var func4 = function(event) {
                            event.stopPropagation();
                            alert("嘿嘿，单击我事件只会触发一次，" + 
                                "即事件不会传递到body元素!");
                        }
                        $("#block35").bind("click", func4);

                                alert(options3[k]);
                            }
    运行
    示例2：两个参数都是对象，且其中某属性也为对象。
                                var options4 = $.deepextend({
                                name: "hcy", age: 22, 
                                info: {address: "gz"}}, 
                                {name: "hcy2", job: "student", 
                                info: {sex: "man", address: "yc"}});
                            for (var l in options4) {
                                var res = options4[l];
                                //如果res是对象，还要继续遍历对象,否则直接输出
                                if (typeof res === "object") {
                                    for (var m in res) {
                                        alert(res[m]);
                                    }
                                } else {
                                    alert(res);
                                }
                            }
    运行

  • $.each(obj, callback)

    概述
    遍历对象或迭代数组，注意在callback函数里this引用当前元素。
    参数
    Object && Function
    示例1：遍历对象
                                $.each({name: "hcy", age: 22, job: "student"}, 
                                function(name, value) {
                                    alert(name + ":" + value);
                            });
    运行
    示例1：迭代数组
                                $.each([1, 2, 3, 4], function(index, value) {//迭代数组
                                alert(index + ":" + value);
                                //alert(this + 3); //4,5,6,7--this引用当前元素
                            });
    运行

  • $.css(elem, name)

    概述
    获取元素的最终样式值
    参数
    Object && String
    示例：读取block40元素样式的width属性值

    p#block40

    $.css($("#block40")[0], "width")
    运行

  • $.data(elem, name, data)

    概述
    读写缓存，如果没有传入data参数，则为读取缓存，如果传入data参数，则为写入缓存。参数依次为要缓存的元素、缓存名称、缓存的数据(可以是任意类型)，返回缓存的数据。
    参数
    Object && String && Attribute
    示例1：写入缓存

    p#block41

    $.data($("#block41")[0], "block41", "嘿嘿，我是block41！")
    运行
    示例2：读取缓存，注意，在没有写入缓存前读取元素的缓存将返回undefined。
    $.data($("#block41")[0], "block41")
    运行

  • $.removeData(elem, name)

    概述
    删除缓存，参数必须对应写入缓存时的参数，即elem、name要一一对应。
    参数
    Object && String
    示例：删除block41元素的缓存
    $.removeData($("#block41")[0], "block41")
    运行
    示例：当再次读取block41元素的缓存时，为undefined。
    $.data($("#block41")[0], "block41")
    运行
